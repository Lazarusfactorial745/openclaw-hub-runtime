# J.A.R.V.I.S. HUD Chat Demo Design

日期：2026-03-13

## 目标

为传统后台聊天界面设计一个可直接双击运行的单文件 `index.html` 高保真 Demo，视觉风格对标《钢铁侠》J.A.R.V.I.S. HUD：深空背景、玻璃拟物、霓虹高亮、信息密集但不压迫。

## 全局视觉语言

- 背景：`#050B14`
- 主高亮：`#00F3FF`
- 点缀/警告：`#FF9D00`
- 玻璃材质：`bg-black/40`、`bg-[#0a0f1e]/60` + `backdrop-blur-md`
- 文本层级：
  - 强文本：`#D9F7FF`
  - 次级文本：`#8CA3B3`
  - 辅助文本：`#5D7385`

## 布局骨架

- `body` 锁定为 `h-screen w-screen overflow-hidden bg-[#050B14]`
- 页面采用左右 `flex`
- 左侧 Sidebar 固定宽度 `w-72`
- 右侧主区 `flex-1 min-w-0 relative flex flex-col`
- 只有中间消息流区域允许滚动
- 底部输入区悬浮于主区底部中央

## 信息密度原则

- 对标 Discord / 开发者终端的紧凑连续流
- 消息间距约 `1rem`
- 通过材质、边框发光、透明度区分消息类型
- 不通过大面积留白区分角色

## 组件定义

### Sidebar

- 玻璃悬浮材质
- 右侧细青色分割线
- 激活项左侧 2px 发光指示线
- 底部展示模型 / 运行状态

### 顶部状态条

- 低存在感 HUD 状态横梁
- 展示当前视图、系统状态 chips、时间 / 节点信息

### 消息流

- AI 消息：冷青玻璃卡片
- 用户消息：更亮、更明确的青色边缘
- 系统日志：高密度终端诊断面板，允许嵌入 token 风格标签

### 系统诊断面板

- 黑色高密度终端块
- 左侧高亮指示线
- 小字号等宽字体
- 警告信息用橙色强调
- `<qqfile>`、`<qqvoice>` 等标签做成 token 胶囊

### 悬浮输入控制台

- 胶囊形强毛玻璃
- 左侧麦克风待机图标
- 中间文本输入区
- 右侧青色发光发送按钮
- 输入区周围保留足够留白，作为页面视觉锚点

## 动效

- 慢呼吸发光：诊断面板、麦克风、发送按钮
- AI 回复带入场动画与闪烁光标
- 背景装饰仅作弱扫描与环境辉光，不喧宾夺主

## 实现约束

- 单文件 HTML
- 使用 Tailwind CDN
- 可加少量内联 CSS / JS
- 自定义滚动条必须通过 `<style>` 手写
- 页面可直接浏览器打开运行，无需构建工具
