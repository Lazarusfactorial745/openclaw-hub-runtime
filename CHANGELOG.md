# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - Initial public release candidate

### Added

- modular OpenClaw Hub Runtime architecture
- flagship `apps/jarvis-hub/` shell
- `examples/minimal-hub/` portability proof
- OpenClaw Gateway adapter and protocol normalizer
- reconnect and recovery manager foundations
- built-in resume / diagnostics showcase commands
- shared Chinese copy module
- architecture, theming, adapter API, onboarding, and contribution docs

### Notes

This version is positioned as the first strong public reference implementation of the runtime direction, not as the final platform release.
